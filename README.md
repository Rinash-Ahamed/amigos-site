# AMIGOS — Multi Brand Fashion Store

A cinematic black & white fashion website built with the latest tech stack.

## Tech Stack

- **React 18** — UI framework
- **Vite 5** — Lightning fast build tool
- **Framer Motion 11** — Advanced animations
- **GSAP 3** — Premium animation library
- **Lenis** — Buttery smooth scroll
- **React Router 6** — Client-side routing

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Pages

- `/` — Home (Hero, Marquee, Category Grid, Featured Products, Editorial, Brands)
- `/collections` — All products with filter by Men / Women / Kids
- `/about` — Brand story & values

## Design System

- **Typography**: Playfair Display (display) + Cormorant Garamond (body) + Space Mono (labels)
- **Colors**: Pure black `#080808` + warm white `#f5f0eb` + charcoal greys
- **Effects**: Custom cursor, grain overlay, smooth scroll, scroll-triggered reveals, image parallax, marquee
- **Images**: HD Unsplash fashion photography, auto grayscale via CSS
